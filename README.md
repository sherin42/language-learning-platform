# 🌐 Language Learning Platform (LLP)

A web-based language learning platform that allows users to learn new languages through interactive lessons and quizzes. The application uses Firebase as a backend-as-a-service for authentication and real-time database management.

---

## 🚀 Features

### 👤 User Features

* Register and login using Firebase Authentication
* Access lessons (video, audio, and text-based)
* Attempt quizzes to test knowledge
* Track learning progress

### 🛠️ Admin Features

* Add, edit, and delete lessons
* Create and manage quizzes
* Monitor user progress
* Admin dashboard for management

---

## 🧰 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend / Database:** Firebase Firestore
* **Authentication:** Firebase Authentication

---

## 📂 Project Structure


## 🔐 Firebase Setup

1. Create a project in Firebase
2. Enable **Authentication (Email/Password)**
3. Create a **Firestore Database**
4. Add your Firebase config in:

```
firebase.js
```
